# 商店上架文案（简体中文）

## 简短描述（建议 ≤ 132 字）

```text
用你自己的 OpenAI 兼容 API 或本地 dual-read-server，做网页双语对照 / 全文替换翻译。
```

## 详细描述

```text
Dual Read 帮助你用「自己的 API」阅读网页：双语对照或直接替换译文。

功能
• 按需翻译当前页（工具栏、Alt+T；选词可走右键菜单）
• 双语对照 / 替换模式（Alt+M 切换）
• 视口优先翻译，节省 token；动态页面增量补译
• 可选按站点自动翻译规则
• 兼容任意 OpenAI 协议端点，或本地 dual-read-server 缓存代理

明确不做
• 不是 VPN、广告拦截、购物助手或通用 AI 侧栏
• 不会把页面默认发到 Dual Read 云——只发到你配置的地址
• 安装时不索要「所有网站」权限

隐私
• 默认零产品遥测
• 远程接口必须 HTTPS（仅 localhost 允许 HTTP）
• API 密钥和自定义 HTTP 请求头仅保存在当前设备；除非明确勾选，否则不会出现在导出文件中
• 完整政策：https://github.com/skys-mission/dual-read/blob/main/PRIVACY.md

三步开始
1. 安装后打开 Options
2. 填写 API 地址与 Key（或指向 http://127.0.0.1:8080/v1）
3. 打开网页 → 点击翻译

开源（Apache-2.0）：https://github.com/skys-mission/dual-read
```

## 分类
