Data & privacy

Translated page text is sent only to your configured API. API keys and custom headers stay on this device; non-secret preferences may sync with your browser account.

  • Visible page text is sent only to the API endpoint you configure.
  • API keys and custom HTTP headers stay on this device and are not browser-synced.
  • Dual Read does not operate a translation cloud of its own.
